⚠️ Safari/iOS CORS restriction

Safari blocks API calls from files opened directly (file://). You need to serve the file from a local web server instead.

On your Mac, open Terminal and run:

cd ~/Downloads && python3 -m http.server 8080

Then open http://localhost:8080/vietnamese_reader.html in Safari.

Keep the Terminal window open while using the app. Press Ctrl+C to stop when done.
